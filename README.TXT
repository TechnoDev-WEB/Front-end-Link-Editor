link text, making it a convenient solution for website content updates.

== Description ==

This plugin provides a convenient solution for those who want to modify links within their
WordPress posts and from the frontend of their page. With the user-friendly hover-activated form editing
links has never been easier. The form  allows for the modification of the link, link
 text and checkbox, making it a simple yet effective tool.

This plugin is designed to help WordPress users modify links within their posts and 
from the frontend of their pages with ease. The plugin is user-friendly and provides a convenient
solution for website posts link updates.The user must be logged in to edit the links. When the user
hovers over the link, a box will appear, which can be clicked to reveal a form above an overlay. 
The form allows the user to edit the link, link text and the option for the link to be opened
in a new tab, which are the only editable options.

== Installation ==
1. Download the plugin files from the repository.
2. Log in to your WordPress admin dashboard.
3. Navigate to the "Plugins" section.
4. Click "Add New" and then "Upload Plugin."
5. Select the plugin file and click "Install Now."
6. Activate the plugin once it has been installed.

 == Usage ==
1. Log in to your WordPress site.
2. Navigate to the post or page where you want to modify a link.
3. Hover over the link to reveal the box.
4. Click the box to reveal the form above an overlay.
5. Use the form to edit the link and link text.
6. Click "Update" to apply the changes.


== Frequently Asked Questions ==

1. Can I edit links within posts that have already been published?

Yes, you can edit links from published posts using the “Front-end Link Editor” plugin.
Simply navigate to the post in the frontend of your site and follow the steps from above.

2. What happens if I have HTML tags within my link text?

If you have html tags in the text of your link, you will not have the option to modify the text
only the link can be modified in this case. The same happens with images, only the link can be modified.

3. What should I do if I encounter an issue with the “Front-end Link Editor” plugin?

If you encounter any issues with the “Front-end Link Editor” plugin, please try 
deactivating and then reactivating the plugin from your WordPress dashboard. If the issue persists, check for
compatibility issues with other plugins or themes you are using. If you are still experiencing difficulties, contact us
for further assistance.
